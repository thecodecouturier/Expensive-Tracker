# 🚀 Quick Start Guide

## البدء السريع (3 دقائق فقط!)

### الخطوة 1: تشغيل السيرفر
```bash
npm start
```

### الخطوة 2: افتح المتصفح
```
http://localhost:3000/register.html
```

### الخطوة 3: أنشئ حساب
```
Name: Your Name
Email: your@email.com
Password: 123456
```

### الخطوة 4: استخدم التطبيق!
- ستُحوَّل تلقائياً للواجهة الرئيسية
- أضف مصروفاتك
- حدد ميزانيتك الشهرية
- شاهد التقارير

---

## 📝 ملاحظات مهمة:

### تسجيل الدخول في المرة القادمة:
```
http://localhost:3000/login.html
```

### الخروج:
اضغط زر 🚪 في الزاوية العلوية

---

## 🎯 المميزات الرئيسية:

✅ **كل مستخدم له حساب منفصل** - بياناتك خاصة بك فقط
✅ **ميزانية لكل شهر** - حدد ميزانية مختلفة كل شهر
✅ **تقارير شهرية** - شاهد مصروفاتك حسب الفئات
✅ **آمن تماماً** - كلمات مرور مشفرة و JWT tokens

---

## 🐛 مشاكل؟

### السيرفر لا يعمل:
```bash
# أوقف العمليات السابقة
lsof -ti:3000 | xargs kill -9

# شغل مرة أخرى
npm start
```

### نسيت كلمة المرور:
- للأسف، لا يوجد استعادة كلمة مرور حالياً
- أنشئ حساب جديد
