🚀 ابدأ النشر في 5 خطوات فقط:
═══════════════════════════════════════════════════════════════

1️⃣  MongoDB Atlas (5 دقائق)
   └─ سجل وأنشئ Cluster مجاني
   └─ احصل على Connection String

2️⃣  رفع على GitHub (2 دقائق)
   git init
   git add .
   git commit -m "Ready for deployment"
   git push

3️⃣  Render - Backend (10 دقائق)
   └─ سجل في Render.com
   └─ أنشئ Web Service
   └─ أضف MONGO_URI في Environment Variables

4️⃣  تعديل Frontend (3 دقائق)
   └─ افتح public/script.js
   └─ غيّر API_BASE_URL برابط Render

5️⃣  Netlify - Frontend (5 دقائق)
   └─ اذهب إلى Netlify
   └─ اسحب مجلد public
   └─ انتهى!
